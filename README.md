# Medical Clinic Management System
Team members: Maja Rusek, Stanislav Volovenko, Vladyslav Shyshnov, Joshua Maru
System name: Medical Clinic Management System
One-sentence goal: Provide a console-based system to manage clinic patients, appointments, staff, and billing to support course requirements and classroom demonstrations.
**Short Description:
The Medical Clinic Management System is a basic console program that helps manage patients, appointments, staff, and billing. It’s designed for class use to show how a small clinic’s information system could work in a clear and easy-to-understand way.
